#  3. Análise por Estrutura / Desafio
**Lista simples:**

- Prompt enviado à IA:
```
Faça um codigo em C que crie uma lista simplesmente encadeada de inteiros, Insera em ordem crescente, Remova um elemento dado um valor e Busque um valor e retornar o ponteiro para o nó (ou NULL).
```
- Versão final do grupo: Não houve muitas mudanças, somente adicionamos a função para usar o free para liberar a memória.

**Lista Dupla:**

- Prompt enviado à IA:
```
Faça um codigo em C que crie uma lista duplamente encadeada de inteiros, Insira no início e no final, Remova do início e do final, Percorrer a lista nos dois sentidos (head → tail, tail → head).
```
- Versão final do grupo: Não houve muitas mudanças, somente adicionamos a função para usar o free para liberar a memória.

**Lista Circular:**

- Prompt enviado à IA:
```
Faça um codigo em C que crie uma lista circular simplesmente encadeada  de inteiros, crie uma função para avançar “k” posições a partir de um nó e crie uma função para remover um nó específico da lista circular mantendo a circularidade.
```
- Versão final do grupo: Não houve muitas mudanças, somente adicionamos a função para usar o free para liberar a memória.

**Pilha(stack):**

- Prompt enviado à IA:
```
Faça um codigo em C que implemente uma pilha usando lista simplesmente encadeada, Operações: push, pop, top, isEmpty e faça um teste verificando se uma expressão com parênteses ()[]{} está balanceada.
```
- Versão final do grupo: Implementamos a funcão de liberar a memoria usando o free, e ajustamos a função verificar se uma expressão está balanceada

**Fila(queue):**

- Prompt enviado à IA:
```
Faça um codigo em C que Implemente uma fila usando lista encadeada, operações: enqueue, dequeue, front, isEmpty e faça um teste simulando atendimento de clientes com chegada em ordem e saída em FIFO.
```
- Versão final do grupo: Não houve muitas mudanças, somente adicionamos a função para usar o free para liberar a memória.
